# Webpack 4 simple start kit

Webpack4 + Babel 7 + Stage-1 / Stage-2 / Stage-3 plugins + Pug + SCSS

### Installing

```
npm i

```

### Running
For development:

```
npm run dev

```

For production:

```
npm run build

```
